# Azure-
# ☁️ Azure Data Fundamentals Project 

This repository contains my Azure workbook completed during the **Level 3 Data Technician Bootcamp**, where I learned the foundations of cloud computing and applied Azure tools to explore, clean, and analyse data in a cloud environment.

Throughout the project, I developed confidence using Azure services to store data, transform it, and understand how cloud platforms support modern data analysis workflows.

---

## 🧠 Key Skills Demonstrated

### ☁️ Understanding Cloud Concepts
I learned core Azure fundamentals, including:

- What the cloud is and why organisations use it  
- Differences between **IaaS, PaaS, and SaaS**  
- Benefits such as scalability, elasticity, and cost‑efficiency  
- Shared responsibility and cloud security principles  

These concepts helped me understand how data is managed, protected, and processed in the cloud.

---

### 🗄️ Working With Azure Storage
I explored how Azure stores and manages data using:

- **Blob Storage** for unstructured data  
- **Containers** and folder structures  
- Uploading, downloading, and organising datasets  
- Understanding redundancy options (LRS, GRS)  

This gave me hands‑on experience preparing data for analysis in a cloud environment.

---

### 🔍 Data Exploration in Azure
Using Azure tools, I learned how to:

- Preview datasets directly in the portal  
- Inspect file formats and metadata  
- Understand schema and data types  
- Identify missing, inconsistent, or poorly structured data  

These skills helped me evaluate data quality before cleaning or analysing it.

---

### 🧹 Data Cleaning & Preparation
I practised preparing data for analysis by:

- Identifying null values and formatting issues  
- Understanding how Azure services handle structured vs unstructured data  
- Exploring how data can be transformed using cloud‑based tools  
- Learning how cloud workflows support repeatable, scalable cleaning processes  

This reinforced the importance of clean, well‑structured data in analytics.

---

### 📊 Data Analysis Concepts in Azure
I learned how Azure supports analytical workflows through:

- Cloud‑based storage for large datasets  
- Integration with tools like Power BI, SQL databases, and Python notebooks  
- Understanding how data flows through a modern cloud pipeline  

This helped me see how Azure fits into real‑world data analysis and business intelligence.
